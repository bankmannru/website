
# Проект Банк Маннру

Добро пожаловать в проект "Банк Маннру"! Этот проект предназначен для предоставления пользователям удобного интерфейса для управления банковскими услугами.

## Описание

Банк Маннру предлагает различные функции, включая:

- **Настройки внешнего вида**: Пользователи могут выбрать тему оформления (по умолчанию, темная, светлая).
- **Выбор языка**: Поддержка нескольких языков, включая русский и английский.
- **Уведомления**: Возможность включения или отключения уведомлений.

## Установка

Чтобы запустить проект локально, выполните следующие шаги:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-репозиторий.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd ваш-репозиторий
   ```
3. Откройте файл `settings.html` в вашем браузере.

## Использование

- Откройте `settings.html` для доступа к настройкам.
- Выберите желаемую тему и язык.
- Настройте уведомления по вашему усмотрению.
